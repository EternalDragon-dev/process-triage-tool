# Sentinel Loop Learning Pack
This directory explains Sentinel Loop as both a working cybersecurity project and a teaching case study for software engineering.

## How to read these docs
- Start with `01_system_overview.md` for the big picture.
- Continue with `02_software_engineering_lessons.md` to understand design and implementation choices.
- Read `03_cybersecurity_dfir_lessons.md` for security/DFIR theory and operational context.
- Use `04_self_correction_design.md` to understand the core innovation (confidence revision loop).
- Review `05_project_progress_timeline.md` for project evolution and engineering milestones.
- Keep `06_glossary.md` open as a terminology reference.

## Intended audience
- Computer science students learning modular design, testability, and maintainability.
- Cybersecurity students learning triage pipelines, evidence handling, and auditability.
- Hackathon judges or reviewers who want a technical walkthrough without reading all source files first.
